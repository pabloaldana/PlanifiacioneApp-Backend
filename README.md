# Planificaciones Data Base

1. Clonar proyecto

2. ```npm i```

3. Clonar el archivo ```.env.template``` y renombrarlo ```.env```

4. Cambiar las variables de entorno

5. Levantar la base de datos

```
docker-composer up -d 
```

6. Levantar: ```npm run start:dev```
